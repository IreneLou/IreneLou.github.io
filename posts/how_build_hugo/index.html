怎麼在Windows環境安裝Hugo並且創建靜態網頁 !

Author included in Note
 2020-05-22  229 words   2 minutes 
Contents
  • 怎麼在Windows環境安裝Hugo
  • 建立個人網站吧!

此篇主要介紹在Windows環境下安裝Hugo，並且用Hugo創建網站發布到Github，產生靜態網頁Github Pages。

怎麼在Windows環境安裝Hugo

Step.0 假設

  1. 會開啟windows的cmd
    (在windows我會安裝 Git Bash，Git Bash能夠使用Linux環境的指令來控制 Windows’ cmd)
  2. Windows 版本為 64-bit
  3. 你要使用的 Website 網址為 <user_name>.github.io
    (因為接下來網頁會發布到Github，所以期待Website的url與你的Github帳號相關)
  4. 你將會使用 D:\Hugo\Sites 當作寫網頁文章的起點 (下一步驟會建立Hugo資料夾)

Step.1 建立Hugo資料夾

需要一個地方來存放你編寫所有hugo網頁的content

  1. 打開 Windows cmd
  2. 創立一個資料夾 D:\Hugo
  3. 創立一個資料夾 D:\Hugo\bin
  4. 創立一個資料夾 D:\Hugo\Sites (存放網頁所有內容，之後很常使用!!)

Step.2 建立Hugo資料夾

  1. 在此網頁 Hugo Release下載目前最新的Hugo版本 " hugo_0.71.0_Windows-64bit.zip "
  2. 下載這包ZIP存在 D:\Hugo\bin 底下，並且解壓縮在此
  3. 解壓縮完會看到 hugo.exe 的執行檔 (如果 .exe的名字不是hugo.exe，需要你幫他改成 hugo.exe)
    另外，hugo.exe 必須放在 D:\Hugo\bin 此資料夾底下喔!!

Step.3 確認安裝完成 & 添加環境變數

  1. 開啟cmd，輸入這行指令 D:\Hugo\bin\hugo version 並且按下 Enter，看到下面的訊息就表示安裝成功啦!
Hugo Static Site Generator v0.70.0-7F47B99E windows/amd64 BuildDate: 2020-05-06T11:17:50Z
  1. 將 D:\Hugo\bin\ 添加進環境變數 Path 中，之後可以直接使用 hugo 下指令
    (例如: 直接輸入 hugo help 就會看到上一點顯示的訊息)

建立個人網站吧!

首先要知道自己想建立什麼網站，我使用Github託管我的用戶網頁，根據Github Pages規則，網站名稱要為 <user_name>.github.io

Step.1 產生網站

cd D:\Hugo\Sites
hugo new site <user_name>.github.io

接下來在Sites目錄底下會看到 <user_name>.github.io 資料夾，進入資料夾可以看到以下結構

|-- archetypes
|-- content
|-- data
|-- layouts
|-- public
|-- static
|-- themes
|-- config.toml

待會使用到的資料夾，先簡單介紹一下!!

content裡面放置網站的文章，static存放靜態資料如圖片、CSS，themes存放網站的主題架構，config.toml為網站的配置屬性文件(網站整體呈現靠它!!)

Step.2 建立第一篇文章!

我們先進到<user_name>.github.io資料夾底下，新增一篇文章

cd <user_name>.github.io
hugo new posts/my_fist_article.md

Hugo會在content底下新增posts資料夾，並創建 my_fist_article.md文件，打開my_fist_article.md會看到裡面有一些內容

---
title: "my_fist_article"
date: 2020-05-22T10:21:26+08:00
draft: true
---

這段包起來的區域是整個網頁Toml配置訊息，叫做 front matter，其中 draft為true時，網站不會發布這篇文章，除了這三項還可以自己添加屬性，以下就是我調整過後的front matter

---
title: "my_fist_article"
date: 2020-05-22T10:21:26+08:00
draft: false
tags: ['hugo']
categories: ['note']
---

然後再把想寫的內容添加在 --- front matter區域底下，一篇文章就完成啦!!

Step.3 添加主題

Hugo主題在這裡可以找到很多模板，我自己是選擇 LoveIt，進入themes目錄然後下載主題

cd themes
git clone https://github.com/dillonzq/LoveIt.git

然後修改 <user_name>.github.io 資料夾底下的config.toml，讓網頁架構符合你的主題，而自己想要在網站添加什麼屬性也是在config.toml做調整
(我使用的LoveIt必須符合他的config.toml格式，所以自己選擇主題時如果沒辦法執行網頁，檢查一下config.toml有沒有格式問題)

baseURL = "http://IreneLou.github.io/"
defaultContentLanguage = "en"
languageCode = "en"
title = "Ciao Ling's 部落格  U・x・U"

Step.4 本地測試

現在到<user_name>.github.io目錄來啟動網頁

cd <user_name>.github.io
hugo server

如果執行成功，可以到 http://localhost:1313/ 看網頁了!!

Step.5 發布到 Github

如果本地測試沒有問題，就可以發布到Github上，讓Github控管你的網頁

首先，建立static版網頁，指令很簡單

hugo -D

編譯後的output會在public目錄中，現在到browser Github創建Repository，名字為<user_name>.github.io，創完後，回到本地端進入public目錄

cd public
git init
git add .
git commit -m "Initial commit."
git remote add origin git@github.com:IreneLou/IreneLou.github.io.git
git push -u origin master

稍等片刻，執行<user_name>.github.io網址，你就可以看到網頁了!!!
