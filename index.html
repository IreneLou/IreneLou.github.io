怎麼在Windows環境安裝Hugo並且創建靜態網頁 !

Author published on 2020-05-22 included in Note
此篇主要介紹在Windows環境下安裝Hugo，並且用Hugo創建網站發布到Github，產生靜態網頁Github Pages。 怎麼在Windows環境安裝Hugo Step.0 假設 會開啟windows的cmd (在windows我會安裝 Git Bash，Git Bash能夠使用Linux環境的指令來控制 Windows’ cmd) Windows 版本為 64-bit 你要使用的 Website 網址為 <user_name>.github.io (因為接下來網頁會發布到Github，所以期待Website的url與你的Github帳號相關) 你將會使用 D:\Hugo\Sites 當作寫網頁文章的起點 (下一步驟會建立Hugo資料夾) Step.1 建立Hugo資料夾 需要一個地方來存放你編寫所有hugo網頁的content 打開 Windows cmd 創立一個資料夾 D:\Hugo 創立一個資料夾 D:\Hugo\bin 創立一個資料夾 D:\Hugo\Sites (存放網頁所有內容，之後很常使用!!) Step.2 建立Hugo資料夾 在此網頁 Hugo Release下載目前最新的Hugo版本 " hugo_0.71.0_Windows-64bit.zip " 下載這包ZIP存在 D:\Hugo\bin 底下，並且解壓縮在此 解壓縮完會看到 hugo.exe 的執行檔 (如果 .exe的名字不是hugo.exe，需要你幫他改成 hugo.exe) 另外，hugo.exe 必須放在 D:\Hugo\bin 此資料夾底下喔!! Step.3 確認安裝完成 & 添加環境變數 開啟cmd，輸入這行指令 D:\Hugo\bin\hugo version 並且按下 Enter，看到下面的訊息就表示安裝成功啦!
